## SQL Injection
### Types of SQL Injection

* In Band SQL Injection
* Blind SQL Injection
* OutOfBand Injection

### Inband SQL Injection Points

- **`Login Form`**
- **`Url Search Parameters`**
- **`Url id values`**
- **`Cookie Values`**
- **`XML encoded values like <storeId>**</storeId>`**
- **`Form Input Sections`**
- **`File Upload Sections`**
